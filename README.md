# aluraplay
Um simples CRUD em MVC com PDO Respository, serviço de login e upload de arquivos.

## Tecnologias utilizadas
- HTML5
- CSS3
- PHP 8.2
- Composer para utilizar autoload
- Banco de dados Sqlite
